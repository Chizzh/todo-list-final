# Файлы для итогового задания

В директории `tests` находятся тесты для проверки API, которое должно быть реализовано в веб-сервере.

Директория `web` содержит файлы фронтенда.

Директория `pkg/api` содержит файлы с обработчиками API запросов.

Директория `pkg/db` содержит файлы для работы с базой данных SQLite.

Директория `pkg/server` содержит файлы для запуска и настройки сервера.

Задания повышенной трудности не выполнялись.

Команда для локального запуска: `go run .`
